# TreeCare Companion MVP

This is a lightweight functional MVP for a homeowner tree-care app.

## What it does
- Add and manage trees
- Store data locally in the browser
- Show a seasonal checklist
- Run a simple risk checker
- Enforce a free-plan limit of 3 trees
- Simulate a premium upgrade
- Install as a basic PWA on supported devices

## How to use it
Open `index.html` in a browser.

For the best installable-app behavior, host it on any simple static host such as:
- Netlify
- Vercel
- GitHub Pages
- Firebase Hosting
